## ComputerVision-Projects 
## (计算机视觉项目整合)

### 项目背景
>该项目是我个人在学习CV过程中，操纵实践代码的整理项目合集，每一个小项目中，都有最新的，基于Python3.6实现相应知识点和小成果的代码。同时有着几乎最详尽的注释。期中有一部分是直接调用第三方库函数查看相应视觉图片处理效果，有一部分是对每种图像处理手段或是算法做了最底层的实现。具体每个小项目中有Readme说明。欢迎了解和补充更新。
